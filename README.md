# Lazo
The browser that doesn’t steal your information or sell it to companies.

This browser was built with one simple idea in mind:
your data belongs to you. Not advertisers. Not analytics companies. Not random third parties hiding in the background.

It does not collect unnecessary personal information.
It does not sell browsing data.
It does not secretly track what you search, watch, or click.

What you do online stays yours.

This project is designed to run smoothly even on low-end computers. If you’re using an older laptop, limited RAM, or weaker hardware, this browser focuses on staying lightweight and responsive instead of bloated and resource-hungry. It aims to use less memory and CPU so your system doesn’t sound like it’s preparing for liftoff.

This is not a giant company project.
It’s built by a single teenager working independently.

That means two things:

It’s made with passion, not profit.

Updates may take longer than commercial browsers.

Since this is a one-person project, development speed depends on time, learning, and real life. There’s no large team behind it. Just one developer building, testing, fixing, improving.

This browser is growing step by step. Each update is focused on stability, performance, and privacy.

It may not have every feature big browsers have yet. But it has something many of them don’t:

Transparency and intent.

No hidden data harvesting.
No background selling of your activity.
No corporate agenda.

Just a lightweight browser built to respect its users.
