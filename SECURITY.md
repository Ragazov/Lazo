# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest  | ✅         |
| 1.2.1   | ✅         |
| < 1.2.1 | ❌         |
| 1.0.0   | ❌         | (Discontinued in 08.06.2026 *dd.mm.yyyy*)

## Reporting a Vulnerability

If you discover a security vulnerability or bug, please report it privately by emailing:

**[gthub0815@gmail.com](mailto:gthub0815@gmail.com)**

Please include:

* A description of the issue
* Steps to reproduce it (if applicable)
* Screenshots or logs that may help investigate the problem

Do not publicly disclose security vulnerabilities until they have been reviewed and addressed.

Thank you for helping improve the security of this project.
